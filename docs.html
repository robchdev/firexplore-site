Getting Started

Qui trovi il flusso base per lavorare in FireXplore: accesso, apertura progetto, query, CRUD e confronto dati tra ambienti.

Contenuti

1. Installazione 2. Login Firebase CLI 3. Progetti e collection 4. JS Query 5. CRUD collection 6. Authentication 7. Compare A/B 8. Casi speciali 9. FAQ

1. Installazione

  1. Apri la pagina Releases e scarica il pacchetto per il tuo sistema.
  2. Installa l'app e avviala normalmente dal sistema operativo.
  3. Alla prima apertura verifica che la connessione internet sia attiva.

2. Login Firebase CLI

FireXplore usa le credenziali del tuo account Firebase CLI. Se la sessione è scaduta, l'app ti riporta alla schermata di login. 

firebase login

Esegui il comando nel terminale e poi riapri FireXplore.

3. Progetti e collection

  • Seleziona un progetto dal pannello sinistro.
  • Apri una collection cliccando sul nome.
  • Usa filtri, ordinamento e layout colonne nella vista tabellare.
  • Le sottocollezioni si aprono dalla colonna dedicata per ogni documento.

4. JS Query

JS Query ti permette di eseguire codice JavaScript sul contesto Firebase Admin SDK e visualizzare il risultato in più formati.

  • Table: vista tabellare dei risultati
  • Tree: navigazione ad albero
  • JSON: payload completo
  • Log: output di debug della query

5. CRUD collection

Dal pulsante CRUD puoi fare insert, update e delete su documenti selezionati. Il payload JSON è editabile e copiabile.

  • Insert: crea nuovi documenti con JSON guidato
  • Update: modifica una o più righe selezionate
  • Delete: elimina documenti selezionati con conferma

6. Authentication

Nella sezione Authentication puoi ispezionare utenti, applicare filtri e lanciare azioni bulk dove supportate (claims, reset password, delete).

7. Compare A/B

Compare confronta due lati (progetto/path A e B) e mostra differenze: cambiato, solo a sinistra, solo a destra, uguale.

Puoi esportare il diff in JSON per audit o review.

8. Casi speciali

  • Firestore Datastore Mode: FireXplore mostra un warning dedicato. In questa modalità alcune operazioni non sono disponibili via API Firestore standard.
  • Sessione scaduta: l'app richiede nuovo login per sicurezza.

9. FAQ

Posso usare FireXplore su più progetti?

Sì, puoi passare da un progetto all'altro e usare compare A/B.

Supporta dataset grandi?

Sì, con paginazione, filtri e strumenti di export/import.

Dove trovo aggiornamenti?

Nel canale Releases, con note versione progressive.
